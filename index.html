How to use this app

This tracker has three screens, switched with the buttons at the bottom of each screen:

  • Run Tracker (screen 1) — plan and follow a Soul Savior run.
  • Champion Record (screen 2) — keep a win/loss tally per starting champion.
  • Help (screen 3) — this page.

Run Tracker

  • Order — set each mid-run region's combat order (1–4) in its Order box. A number you pick is greyed out for the other regions, and the rows re-sort into that order. Astrael is always first, the Lifemother always last.
  • Variant & Central Nodes — for each encounter, pick its boss / wave-set Variant and its Central Nodes from the dropdowns.
  • Info box — click a Battle or Boss row to see its waves and enemy stats below. Hover an enemy name for its ability note; a boss's curse box shows its mutator on hover.
  • Reset grid — clears every selection for the current run (it asks first). It does not affect your Champion Record.

Champion Record

  • Win / Loss — use the −/+ buttons on each row to adjust a champion's counts (they never go below 0).
  • Reorder — drag a row by its grip (⠿) to move it; the new order is saved. The 24 champions fill the left column first, then the right.
  • Highlight — click a champion's name (or work with its row) to highlight it; click the name again to clear it.
  • Reset record — clears all counts and restores the default order (it asks first). It does not affect the Run Tracker.

Everything you enter is saved in your browser and restored when you come back.
