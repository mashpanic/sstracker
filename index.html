How to use this app

This tracker has three screens, switched with the buttons at the bottom of each screen:

  • Run Tracker — plan and follow a Soul Savior run.
  • Win-Loss — keep a win/loss tally per starting champion.
  • Help — this page.

Run Tracker

  • Order — set each mid-run region's combat order (1–4) in its Order box. A number you pick is greyed out for the other regions, and the rows re-sort into that order.
  • Variant — for each encounter, pick its boss / wave-set Variant.
  • Central Nodes — select Central Nodes from the dropdown, or focus a box and press the first letter of a choice. Quickly enter sets such as Magic, Trinkets, Boons by hitting M (Tab) T (Tab) B (Tab).
  • Wave box — click a Battle or Boss row to see its waves. You must select the enemy wave set if required. If no order is selected, a preview of the next unselected Order will show. You must still manually set that order.
  • Reset grid — clears every selection for the current run (it asks first). It does not affect your Win-Loss record.

Win-Loss

  • Win-loss score — each row shows the champion's record as wins-losses (e.g. "7-3").
  • Select — click a champion's row to select it (green highlight); click it again to clear the selection.
  • Adjust — with a champion selected, the grouped W+ / W− / L+ / L− buttons at the bottom change its record (counts never go below 0). They're disabled until a champion is selected.
  • Lap — with a champion selected, ▲ records a win (win +1, green up arrow) and ▼ records a loss (loss +1, red down arrow); the arrow shows just before its win-loss score. Click the same arrow again to undo it (and clear the arrow); clicking the opposite arrow switches the result. Reset Lap clears the arrows for a new lap while keeping the win/loss totals.
  • Reorder — drag a row by its grip (⠿) to move it; the new order is saved. The 24 champions fill the left column first, then the right.
  • Reset order — restores the default champion order without changing any win/loss counts (it asks first).
  • Reset record — clears all win/loss counts but keeps the current order and lap arrows (it asks first). It does not affect the Run Tracker.
  • Reset Lap — clears every champion's lap arrow but keeps the win/loss counts and order (it asks first).

Everything you enter is saved in your browser and restored when you come back.

Backup & transfer

Your data lives only in this browser. Use these to move your Win-Loss record to another browser or device, or to keep a backup. The export is a plain-text JSON file. The Run Tracker is not included — it's for a single run and isn't meant to carry over, and importing never touches an in-progress run.

  • Export data — shows your Win-Loss record as JSON text you can copy or download as a file.
  • Import data — paste that text (or pick a downloaded file) to load it. It replaces your Win-Loss record, then reloads the page.
